## Development

1. Clone the repository:
   ```bash
   git clone https://github.com/roomrys/linalg.git
   ```
2. Run the server:
   (To avoid CORS error)

   ```bash
   python3 -m http.server 8000
   ```

3. Browse to:
   ```bash
   http://[::]:8000/
   ```
